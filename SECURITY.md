# Security Policy

## Scope

The Strategic Labelling Framework is a documentation and annotation system. It contains no executable code that handles user data, authentication, or network connections.

## Reporting a Vulnerability

If you identify a security concern in this repository (for example, a dependency issue or a problematic example script), please open a GitHub Issue describing the concern.

There is no formal disclosure process for this project given its nature, but all reports will be reviewed and responded to promptly.

## Supported Versions

Only the current `main` branch is actively maintained.
